# random-colors
<h1>Generador de colores aleatorios</h1>

<h2>Funciones</h2>

<ul>
  <li>Generar paletas de colores aleatorias</li>
  <li>Bloquear los colores que te gusten para que se sigan generando los demás solamente aleatorios</li>
  <li>Copiar un color al portapapeles haciendo clic en su código</li>
  <li>Guardar en la biblioteca las paletas que más te gusten para reutilizarlas más tarde</li>
  <li>Editar los valores de tono, brillo y saturación para ajustar cada color si lo deseas</li>
  <li>Ir a la biblioteca y volver a usar alguna de las paletas guardadas</li>
  <li>Reiniciar la biblioteca para borrar todos los colores</li>
</ul>

<p>¡A divertirse!</p>
